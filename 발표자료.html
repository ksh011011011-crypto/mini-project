VIP MEMBERSHIP

VIP 멤버십 시스템

2026년 기준 등급별 자동 산정

👑
VIP
연 260,000원 이상
  • ✅ 일반 관람권 [주중] 2매
  • ✅ 스페셜관 관람권 1매
  • ✅ 팝콘M 교환권 1매
  • ✅ 생일 스위트 콤보 1매
👑
VVIP
연 300,000원 이상
  • ✅ A형/B형 선택 관람권 최대 8매
  • ✅ 스페셜관 관람권 2매
  • ✅ 팝콘M + 음료 교환권 2매씩
  • ✅ 생일 스위트 콤보 + 추가 혜택
👑
LVIP
연 360,000원 이상
  • ✅ A형/B형 관람권 최대 12매
  • ✅ 스페셜관 + 매점 혜택 최대
  • ✅ 세현마트 15% 할인
  • ✅ 세현 렌터카 92% 할인

⚙️ 자동 등급 산정 로직 (Member.java)

public void updateGrade() {
  if (totalSpent >= 360000) grade = "LVIP";
  else if (totalSpent >= 300000) grade = "VVIP";
  else if (totalSpent >= 260000) grade = "VIP";
  else grade = "일반";
}
DATABASE DESIGN

데이터베이스 설계

ERD - PostgreSQL (Supabase 연동)

👤 members (회원)
id (PK)BIGINT자동 증가
usernameVARCHAR(50)로그인 ID (unique)
gradeVARCHAR(20)일반/VIP/VVIP/LVIP
total_spentINTEGER누적 결제액 (등급 산정)
🎬 movies (영화)
id (PK)BIGINT자동 증가
titleVARCHAR(100)영화 제목
statusVARCHAR(20)SHOWING/UPCOMING/ARTE
rating_avgDECIMAL(3,2)평균 평점
rank_positionINTEGERTOP 순위
📋 notices (공지사항)
categoryVARCHAR(50)전체/서울/경기 등
is_importantBOOLEAN중요 공지 여부 (!)
view_countINTEGER조회수
🎟️ bookings (예매)
member_id (FK)BIGINT→ members
movie_id (FK)BIGINT→ movies
theater_typeVARCHAR(30)일반관/IMAX 등
seat_infoVARCHAR(20)D-12 형식
total_priceINTEGER최종 결제 금액
⭐ reviews (리뷰)
member_id (FK)BIGINT→ members
movie_id (FK)BIGINT→ movies
ratingINTEGER1~5점
❤️ hearts + 💬 inquiries
heartsunique(member,movie)좋아요 (중복 방지)
inquiries+ faqs1:1문의 / FAQ CRUD

CUSTOMER SERVICE

고객센터 기능

FAQ · 공지사항 · AI 상담톡 · 1:1 문의

📚

FAQ 시스템

  • 9개 카테고리 탭 (영화관이용·스페셜관·L.POINT·회원·멤버십·온라인·할인혜택·관람권·스토어)
  • 질문 클릭 시 아코디언 방식 답변 표시
  • 키워드 검색 + 페이지네이션 (10개씩)
  • 1:1 문의 바로가기 링크
📢

공지사항

  • 카테고리 필터 (전체·영화관·서울·경기/인천)
  • 중요 공지(!) 표시 - 상단 고정
  • 제목/내용 검색 + 페이지네이션
  • 조회수 자동 증가
🤖

AI 상담톡 챗봇

  • 빠른 선택 버튼 (예매/결제·영화관·이벤트·분실)
  • 키워드 기반 자동 응답 (예매·취소·VIP·IMAX 등)
  • 욕설 3회 감지 시 상담 차단 경고
  • 1분 무응답 자동 종료
  • "자동예매" 키워드 → 예매 페이지 연결
✉️

1:1 문의

  • 분류: 영화관·멤버십/회원·예매/결제·이벤트
  • 종류: 문의·칭찬·불만·건의
  • 이메일/SMS 알림 선택
  • 개인정보 수집 동의 항목 포함
  • MY 문의내역 조회 (로그인 시)
