# 세현시네마 (SEHYUN CINEMA) — PRD 요약본 (Claude PPT용)

이 파일은 **발표 PPT**를 Claude 등에 넣을 때 복사하기 쉽게 만든 제품 요약입니다.  
동일 내용 웹 페이지: 서버 실행 후 `http://localhost:8080/prd-claude.html`

---

## 1. 제품 개요

- **이름**: 세현시네마 (SEHYUN CINEMA)
- **유형**: 온라인 영화관 예매·멤버십 웹 (멀티플렉스 UX 참고)
- **목표**: 영화 → 상영관·일시·좌석 → 결제, VIP·쿠폰·가상카드, TMDB 포스터
- **기술**: Java 21, Spring Boot 3.4.x, Thymeleaf, Spring Security, OAuth2(카카오), PostgreSQL(Supabase), Gradle

---

## 2. 핵심 기능

- **메인**: TOP10, 현재상영·개봉예정·아르떼, 검색·더보기, TMDB 포스터
- **영화 상세**: 포스터, 설명, 평점·예매율, 찜, 예매, 관람평(별점+글)
- **예매**: 영화·관·일시·인원·좌석·결제
- **회원**: 등급제, 마이페이지, 예매내역, 쿠폰
- **기타**: 공지·FAQ·문의, AI 상담, 스페셜관(IMAX·4DX 등)

---

## 3. 포스터·데이터

- TMDB 이미지 URL을 DB에 저장
- 기동 시 제목 매칭으로 포스터 동기화, 빈 URL 보정

---

## 4. 보안

- 폼 로그인 + 카카오 OAuth
- 예매·마이페이지는 인증 필요
